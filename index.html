GitHub sync — setup
Uses two separate repos — your app stays public, your personal data stays private.
PUBLIC  ironlog — hosts the app (GitHub Pages)
PRIVATE  ironlog-data — stores your workout data
github.com → Settings → Developer settings → Personal access tokens → Tokens (classic) → Scope: repo → Expiry: No expiration
Step-by-step setup
1
Create a private GitHub repo called ironlog-data (github.com → + → New repository → set Private → Create)
2
Go to your avatar → Settings → Developer settings → Personal access tokens → Tokens (classic) → Generate new token
3
Name it IronLog · Expiry: No expiration · Tick scope: repo · Generate and copy the token
4
Fill in your username, repo names, and token above → tap Connect & sync
